# Game2_in_Unity
Another very easy game to relax.
Rules:
- space = jump
- double space = double jump
- arrows = left and right
- player's objective is to get to the top of the obstacle course
- apple = 1 point
- pineapple = 10 points
- melon = 20 points
- more levels to come :)
